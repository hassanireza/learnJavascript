# Project 05: Multiplication Table Generator

| Difficulty | Estimated Time | Concepts Practiced |
|---|---|---|
| Beginner | 30 minutes | for loops, nested loops, template literals |

## Objective

Generate and print a multiplication table from 1 to 12 for a given number.

## Requirements

- Declare a `const number` to generate the table for, for example `7`
- Use a `for` loop to print each line: `7 x 1 = 7`, `7 x 2 = 14`, and so on through `7 x 12`
- Use a template literal to format each line

## Bonus Challenges

- Use nested for loops to print full multiplication tables for the numbers 1 through 5
- Skip even multiples using `continue`, printing only odd results
