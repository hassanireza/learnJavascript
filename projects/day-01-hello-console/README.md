# Project 01: Hello Console

| Difficulty | Estimated Time | Concepts Practiced |
|---|---|---|
| Beginner | 20 minutes | console.log, comments, running scripts |

## Objective

Create a script that introduces you using only `console.log()` statements, and run it successfully both in the browser console and with Node.js.

## Requirements

- Print your name to the console
- Print one fact about yourself
- Print the message `"I am learning JavaScript with a 30 day course."`
- Add at least one comment explaining what the script does

## Bonus Challenges

- Print today's date using `new Date()`
- Use `console.log()` with multiple comma separated arguments in one call

## Files in This Project

```text
projects/day-01-hello-console/
├── README.md
├── index.html
├── style.css
└── script.js
```

Open `index.html` with the **Live Server** extension, or open `script.js` directly in a Node.js environment, depending on what the lesson requires. Look for `// TODO` comments inside `script.js` to know exactly what to implement.

## Solution Approach

<details>
<summary>Click to reveal a hint if you get stuck (try solving it yourself first)</summary>

Use one `console.log()` call per line of output, and place a `//` comment above the script explaining its purpose. Run it with `node script.js` to confirm it works outside the browser too.

</details>
