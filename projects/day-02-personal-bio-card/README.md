# Project 02: Personal Bio Card

| Difficulty | Estimated Time | Concepts Practiced |
|---|---|---|
| Beginner | 30 minutes | let, const, primitive types, typeof, string concatenation |

## Objective

Use variables of different data types to build and print a short personal bio card to the console.

## Requirements

- Create a `const` for your full name (String)
- Create a `const` for your age (Number)
- Create a `let` for your current city, since it might change (String)
- Create a `const` boolean indicating whether you are currently learning to code
- Print a formatted multi line bio using these variables
- Log the `typeof` of each variable

## Bonus Challenges

- Add a variable that is intentionally `null` to represent something unknown, such as your future job title
- Try reassigning a `const` and observe the error in your console
